Getting Started

Welcome to the odeFlow Mastery API. Our API enables you to integrate our powerful project management features directly into your applications. 

curl -X GET "https://api.codeflow.dev/v1/projects"
-H "Authorization: Bearer YOUR_API_KEY"

Authentication

All API requests require authentication using a Bearer token. You can obtain your API key from your dashboard settings.

Endpoints

Projects

Manage your projects through the API.

GET /v1/projects

List all projects

POST /v1/projects

Create a new project

Tasks

Manage tasks within your projects.

GET /v1/tasks

List all tasks

POST /v1/tasks

Create a new task

Error Handling

Our API uses conventional HTTP response codes to indicate success or failure of requests.

200 - OK

Request successful

400 - Bad Request

Invalid request parameters

401 - Unauthorized

Invalid or missing API key

Rate Limits

To ensure fair usage, our API implements rate limiting based on your subscription plan.

Plan Rate Limit
Free 100 requests/hour
Pro 1000 requests/hour
Enterprise Custom limits
